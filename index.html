Integration Guide

Getting Started

  1. Click "Create New Project" on the home page
  2. Load a video file using the file input or drag & drop
  3. Add subtitles by clicking the "Add" button in the subtitle panel
  4. Adjust timing by clicking on each subtitle entry
  5. Export your completed subtitles as .SRT file

Video Format Support

Supported formats: MP4, WebM, OGG, MOV

Export Format

Subtitles are exported in standard SRT format compatible with most video players and platforms.
